# ProgramViz 进度跟踪

最后更新：2026-07-15

本文档记录已经落地的能力、当前限制和后续工作。复选框表示代码状态，不代表发布承诺。

## 当前状态

ProgramViz 已具备可运行的 Qt 6 原型：可以扫描工程、统计代码行数、绘制文件树状图，并提供节点详情、外观设置和基础 Git 历史可视化。JPG 与 Git 演进 GIF 导出已经接入，但项目仍处于开发阶段，尚未完成跨平台验证、性能优化、安装打包和许可证配置。
